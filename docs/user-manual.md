# 차량 운행일지 — 사용자 매뉴얼

> 삼우에레코주식회사 | 경영지원그룹

---

## 📱 운전자 사용법

### 1. 로그인
- 지급받은 이메일과 비밀번호로 로그인
- 모바일 기기에서 앱 설치 배너가 뜨면 "설치"를 눌러 홈화면에 추가

### 2. 운행 시작
1. 홈 화면 → **"운행 시작"** 탭
2. 차량 선택 → 출발지 입력 → 계기판 km 입력
3. 업무 목적 선택 → **"🚗 운행 시작"** 버튼

### 3. 운행 완료
1. 목적지 도착 후 홈 화면 → **"도착 등록하기"** 배너 탭
2. 목적지 입력 → 도착 계기판 km 입력
3. 통행료가 있으면 입력 (빠른 선택 버튼 활용)
4. **"🏁 운행 완료"** 버튼

### 4. 월말 제출
1. **내 기록** 탭 → 해당 월 선택
2. 완료된 기록 확인 후 **"전체 제출"** 버튼
3. 관리자 승인 대기 (이메일 알림 수신)

### 5. 오프라인 사용
- 네트워크 없는 곳에서도 출발/도착 등록 가능
- 상단에 노란 배너가 뜨면 오프라인 상태
- 온라인 복귀 시 자동으로 서버에 동기화됨

## 💻 관리자 사용법

### 1. 대시보드
- 이번 달 KPI (운행 건수 · 거리 · 통행료 · 승인대기)
- 월별 추이 차트, 차량별 비교 차트
- 운행 기록 변경 시 **실시간 자동 갱신**

### 2. 운행 현황
- **운행 현황** 메뉴 → 기간·차량·운전자·상태 필터
- 행 클릭 → 상세 보기 / 인라인 승인 처리

### 3. 승인 관리
- **승인 관리** 메뉴 → 승인 대기 목록
- **검토하기** 버튼 → 내용 확인 → 승인 / 반려 (코멘트 선택 입력)
- 처리 결과가 운전자에게 이메일로 자동 발송됨

### 4. 보고서 출력
- **보고서 출력** 메뉴 → 연도 · 월 · 차량 선택
- **📥 Excel 다운로드** — 세무 증빙용 국세청 양식 (Sheet1: 상세, Sheet2: 집계)
- **📄 PDF 미리보기** — 서명란 포함 A4 가로 출력용

### 5. 차량 · 운전자 관리
- **차량 관리** / **운전자 관리** 메뉴에서 등록·수정·삭제
- 운행 기록이 있는 차량·운전자는 **비활성화** 처리 (기록 보존)

## ⚙️ 초기 설정 순서 (관리자)

```
1. Supabase 대시보드 → SQL Editor → 001_init_schema.sql 실행
2. Auth → 관리자 계정 생성
3. 테이블 → user_roles → role = 'admin' 으로 변경
4. 차량 관리 → 영업용 차량 등록
5. 운전자 관리 → 직원 등록 → 각자 로그인 후 계정 연결
```

## ❓ 자주 묻는 질문

| 질문 | 답변 |
|------|------|
| 도착 km를 잘못 입력했어요 | 기록 탭 → 해당 기록 → 수정 (제출 전 draft 상태만 가능) |
| 승인 요청 후 수정이 안 돼요 | 관리자에게 반려 요청 → 반려 후 수정 가능 |
| 통행료 영수증은 어떻게 첨부하나요 | 현재 버전은 금액만 입력, 영수증은 별도 보관 |
| 다른 직원 기록이 보여요 | 관리자에게 문의 (역할 설정 문제) |
| 앱이 느려요 | 오프라인 동기화 중일 수 있음 — 잠시 후 재시도 |
